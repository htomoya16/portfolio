# Portfolio Site

このリポジトリは堀田智哉（Hotta Tomoya）のポートフォリオサイトです。  
Next.js + Tailwind CSS で構築しており、以下のURLで公開しています。

- 本番: https://htomoya16.dev/
- ソース: `portfolio-site/` ディレクトリ配下

ローカルで確認する場合:
```bash
cd portfolio-site
pnpm install
pnpm dev
```

主なセクション: About / Skills / Projects / Research / Contact  
デザインは青×白のシンプルなトーンで、バックエンド開発とVR/Hapticsの実績を掲載しています。
